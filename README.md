## 同一个版本，为什么有不同的安装包？       
这些安装包分别对应不同的wine版本，不同的wine版本兼容性不同，性能可能也不同(可以忽略不计)，一般用wine4即可，喵玉汉化补丁用不了的用wine6。       
几MB的安装包是`非直装版`的，一般不用这个版本。
